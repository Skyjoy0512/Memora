## 第1部 プロジェクト基盤

### 1.1 開発環境

本プロジェクトは iOS 18 以上を最小デプロイターゲットとし、Xcode 16 系でビルドする。言語は Swift 6 を採用し、Strict Concurrency を有効にする。UIフレームワークは SwiftUI を主体とし、AVFoundation など UIKit 依存が必須な箇所のみ UIViewRepresentable でブリッジする。パッケージ管理は Swift Package Manager に統一し、CocoaPods / Carthage は使用しない。

アーキテクチャは TCA（The Composable Architecture）を採用する。各画面を Feature 単位の Reducer として分離し、State / Action / Reducer / Effect の4要素で構成する。依存注入は TCA の DependencyKey プロトコルを利用し、テスト時にはモックへ差し替え可能にする。

CI は GitHub Actions を前提とし、push ごとに `xcodebuild test` を実行する。Codex / Claude Code が生成したコードもこのパイプラインで検証する。

### 1.3 依存パッケージ

TCA は `swift-composable-architecture` 1.17 以上を使用する。認証では `AuthenticationServices`（Apple 標準）と `GoogleSignIn-iOS` 7.x を使用する。課金は `StoreKit 2` をネイティブで使用し、RevenueCat は Phase 1 では導入しない。ローカル文字起こしには `Speech` フレームワーク（SFSpeechRecognizer）を使用し、端末ローカルで完結させる。広告は `GoogleMobileAds` 11.x を使用する。ネットワーク層は `Foundation.URLSession` をベースに薄いラッパーを自作する。テストでは `swift-snapshot-testing` を使用してUI回帰テストを行う。

### 2.3 Liquid Glass モディファイア

液体ガラスの質感は、背景ブラー、半透明オーバーレイ、ソフトシャドウの3層で構成する。

```swift
// LiquidGlassModifier.swift
import SwiftUI

struct LiquidGlassModifier: ViewModifier {
    var cornerRadius: CGFloat = 16
    var opacity: Double = 0.72
    var shadowRadius: CGFloat = 12

    func body(content: Content) -> some View {
        content
            .background(
                .ultraThinMaterial,
                in: RoundedRectangle(cornerRadius: cornerRadius, style: .continuous)
            )
            .overlay(
                RoundedRectangle(cornerRadius: cornerRadius, style: .continuous)
                    .fill(Color.white.opacity(opacity))
                    .blendMode(.overlay)
            )
            .overlay(
                RoundedRectangle(cornerRadius: cornerRadius, style: .continuous)
                    .stroke(Color.white.opacity(0.3), lineWidth: 0.5)
            )
            .shadow(color: MemoraColor.shadowMedium, radius: shadowRadius, x: 0, y: 4)
    }
}

extension View {
    func liquidGlass(
        cornerRadius: CGFloat = 16,
        opacity: Double = 0.72,
        shadowRadius: CGFloat = 12
    ) -> some View {
        modifier(LiquidGlassModifier(
            cornerRadius: cornerRadius,
            opacity: opacity,
            shadowRadius: shadowRadius
        ))
    }
}
```

### 2.5 共通コンポーネント仕様

**FABMenu** は画面右下に配置する丸型ボタンで、タップすると「端末録音」「ファイルインポート」の2選択肢が扇状に展開する。ボタンの直径は 56pt、展開時の子ボタンは 44pt とする。展開アニメーションは spring(response: 0.35, dampingFraction: 0.75) で制御する。背景は liquidGlass を適用する。

**ToastOverlay** は画面上部から下降するバナー型の通知で、失敗通知に使用する。高さは自動伸縮、左にアイコン、中央にメッセージ、右に閉じるボタンを配置する。表示時間は 4 秒で自動消去する。liquidGlass を適用し、角丸は 12pt とする。

**EmptyStateView** は中央にイラスト（またはSFSymbol）、その下にタイトルと説明文、さらにその下にプライマリアクションボタンを配置する構造を持つ。Files / Projects / ToDo の各空状態で再利用する。

**SkeletonView** は生成処理中のプレースホルダーで、角丸の矩形が shimmer アニメーション（左から右への光沢移動）を繰り返す。高さは使用箇所に応じて可変とする。

**LiveRecordingBanner** は録音中に Files 一覧の最上部に表示される横長バナーで、赤い録音インジケータドット、経過時間、波形インジケータ、停止ボタンを横一列に配置する。タップするとRecordingViewへ遷移する。

## 第3部 データ層

### 3.1 永続化方針

Phase 1 のローカル保存には SwiftData を採用する。ModelContainer は MemoraApp の起動時に1つだけ生成し、環境経由ですべてのビューに渡す。マイグレーションは VersionedSchema で管理し、スキーマ変更時には MigrationPlan を必ず定義する。

音声ファイル本体と添付ファイルは FileManager の Application Support ディレクトリ配下にサブディレクトリを切って保存する。SwiftData には相対パスのみ記録し、ベースパスは起動時に解決する。

### 3.3 Repository パターン

各エンティティに対して Repository プロトコルを定義し、SwiftData の ModelContext を直接ビューや Reducer から触らせない。これによりテスト時のモック化とデータアクセスの一元管理を実現する。

```swift
// AudioFileRepository.swift
import Foundation
import SwiftData

protocol AudioFileRepositoryProtocol: Sendable {
    func fetchAll() async throws -> [AudioFile]
    func fetch(id: UUID) async throws -> AudioFile?
    func save(_ file: AudioFile) async throws
    func delete(_ file: AudioFile) async throws
    func fetchByProject(_ projectId: UUID) async throws -> [AudioFile]
}

final class AudioFileRepository: AudioFileRepositoryProtocol {
    private let modelContainer: ModelContainer

    init(modelContainer: ModelContainer) {
        self.modelContainer = modelContainer
    }

    @ModelActor
    actor Worker {
        func fetchAll() throws -> [AudioFile] {
            let descriptor = FetchDescriptor<AudioFile>(
                sortBy: [SortDescriptor(\.recordedAt, order: .reverse)]
            )
            return try modelContext.fetch(descriptor)
        }

        func fetch(id: UUID) throws -> AudioFile? {
            let descriptor = FetchDescriptor<AudioFile>(
                predicate: #Predicate { $0.id == id }
            )
            return try modelContext.fetch(descriptor).first
        }

        func save(_ file: AudioFile) throws {
            modelContext.insert(file)
            try modelContext.save()
        }

        func delete(_ file: AudioFile) throws {
            modelContext.delete(file)
            try modelContext.save()
        }

        func fetchByProject(_ projectId: UUID) throws -> [AudioFile] {
            let descriptor = FetchDescriptor<AudioFile>(
                predicate: #Predicate { $0.project?.id == projectId },
                sortBy: [SortDescriptor(\.recordedAt, order: .reverse)]
            )
            return try modelContext.fetch(descriptor)
        }
    }

    func fetchAll() async throws -> [AudioFile] {
        let worker = Worker(modelContainer: modelContainer)
        return try await worker.fetchAll()
    }

    func fetch(id: UUID) async throws -> AudioFile? {
        let worker = Worker(modelContainer: modelContainer)
        return try await worker.fetch(id: id)
    }

    func save(_ file: AudioFile) async throws {
        let worker = Worker(modelContainer: modelContainer)
        try await worker.save(file)
    }

    func delete(_ file: AudioFile) async throws {
        let worker = Worker(modelContainer: modelContainer)
        try await worker.delete(file)
    }

    func fetchByProject(_ projectId: UUID) async throws -> [AudioFile] {
        let worker = Worker(modelContainer: modelContainer)
        return try await worker.fetchByProject(projectId)
    }
}
```

他のエンティティ（Transcript, MeetingNote, Project, TodoItem, Attachment, ProcessingJob）も同一のパターンで実装する。各 Repository は TCA の `DependencyKey` として登録し、Reducer から `@Dependency(\.audioFileRepository)` で取得する。

### 4.1 認証フロー

AuthService は Sign in with Apple と Sign in with Google の2つのプロバイダをサポートする。内部的には Firebase Authentication を使用し、各プロバイダの認証トークンを Firebase credential に変換して統一的に管理する。

```swift
// AuthService.swift
import AuthenticationServices
import GoogleSignIn

protocol AuthServiceProtocol: Sendable {
    func signInWithApple(credential: ASAuthorizationAppleIDCredential) async throws -> AuthUser
    func signInWithGoogle(presentingWindow: UIWindow) async throws -> AuthUser
    func signOut() async throws
    func currentUser() async -> AuthUser?
    func observeAuthState() -> AsyncStream<AuthUser?>
}

struct AuthUser: Equatable, Sendable {
    let uid: String
    let email: String?
    let displayName: String?
    let provider: AuthProvider
}

enum AuthProvider: String, Codable, Sendable {
    case apple  = "apple"
    case google = "google"
}
```

AuthReducer は以下の State / Action を持つ。

```swift
@Reducer
struct AuthReducer {
    @ObservableState
    struct State: Equatable {
        var isLoading: Bool = false
        var user: AuthUser? = nil
        var error: String? = nil
    }

    enum Action {
        case onAppear
        case signInWithAppleTapped
        case signInWithAppleResponse(Result<AuthUser, Error>)
        case signInWithGoogleTapped
        case signInWithGoogleResponse(Result<AuthUser, Error>)
        case signOutTapped
        case signOutResponse(Result<Void, Error>)
        case authStateChanged(AuthUser?)
    }

    @Dependency(\.authService) var authService

    var body: some ReducerOf<Self> {
        Reduce { state, action in
            switch action {
            case .onAppear:
                return .run { send in
                    for await user in authService.observeAuthState() {
                        await send(.authStateChanged(user))
                    }
                }
            case .signInWithAppleTapped:
                state.isLoading = true
                state.error = nil
                return .none // Apple Sign In は ASAuthorizationController のデリゲート経由
            case .signInWithAppleResponse(.success(let user)):
                state.isLoading = false
                state.user = user
                return .none
            case .signInWithAppleResponse(.failure(let error)):
                state.isLoading = false
                state.error = error.localizedDescription
                return .none
            case .signInWithGoogleTapped:
                state.isLoading = true
                state.error = nil
                return .run { send in
                    // UIWindow は Environment 経由で取得
                    // 実装時に適切に対応
                }
            case .signInWithGoogleResponse(.success(let user)):
                state.isLoading = false
                state.user = user
                return .none
            case .signInWithGoogleResponse(.failure(let error)):
                state.isLoading = false
                state.error = error.localizedDescription
                return .none
            case .signOutTapped:
                return .run { send in
                    try await authService.signOut()
                    await send(.signOutResponse(.success(())))
                }
            case .signOutResponse(.success):
                state.user = nil
                return .none
            case .signOutResponse(.failure(let error)):
                state.error = error.localizedDescription
                return .none
            case .authStateChanged(let user):
                state.user = user
                return .none
            }
        }
    }
}
```

### 5.1 オンボーディング（OB-01〜04）

4ページのページングスクロールで構成する。各ページには中央にイラスト、下部にタイトルと説明文を配置する。最下部に「次へ」ボタンと現在のページインジケータ（ドット）を配置する。最終ページのボタンテキストは「始める」とする。

```swift
@Reducer
struct OnboardingReducer {
    @ObservableState
    struct State: Equatable {
        var currentPage: Int = 0
        let totalPages: Int = 4
    }

    enum Action {
        case nextTapped
        case pageChanged(Int)
        case completed
    }

    var body: some ReducerOf<Self> {
        Reduce { state, action in
            switch action {
            case .nextTapped:
                if state.currentPage < state.totalPages - 1 {
                    state.currentPage += 1
                    return .none
                } else {
                    return .send(.completed)
                }
            case .pageChanged(let page):
                state.currentPage = page
                return .none
            case .completed:
                return .none // 親 Reducer がハンドル
            }
        }
    }
}
```

OnboardingView は `TabView` の `.tabViewStyle(.page)` を使用し、`selection` を State.currentPage にバインドする。背景は `MemoraColor.surfacePrimary`、ボタンは黒背景白文字の pill 形状（高さ 50pt、角丸 25pt）とする。

```swift
                            try await audioFileRepository.delete(file)
                        }
                        await send(.onAppear) // reload
                    }
                case .rename, .addToProject, .share:
```

### 5.6 録音（REC-01）

RecordingView は全画面のモーダルとして表示する。背景は `MemoraColor.surfacePrimary`。中央に大きな経過時間ラベル（`MemoraTypography.largeTitle`）を配置する。その下に波形ビジュアライザー（AVAudioRecorder の `averagePower(forChannel:)` からリアルタイムに描画する横長のバー群）を配置する。最下部に録音停止ボタン（赤い丸型、直径 72pt）を配置する。

```swift
@Reducer
struct RecordingReducer {
    @ObservableState
    struct State: Equatable {
        var isRecording: Bool = false
        var duration: TimeInterval = 0
        var audioLevels: [Float] = []
        var savedFileId: UUID? = nil
    }

    enum Action {
        case onAppear
        case startRecording
        case stopRecording
        case timerTicked
        case audioLevelUpdated(Float)
        case recordingSaved(UUID)
        case recordingFailed(String)
        case dismiss
    }

    @Dependency(\.audioRecorder) var audioRecorder
    @Dependency(\.continuousClock) var clock

    var body: some ReducerOf<Self> {
        Reduce { state, action in
            switch action {
            case .onAppear:
                return .send(.startRecording)
            case .startRecording:
                state.isRecording = true
                state.duration = 0
                return .merge(
                    .run { send in
                        try await audioRecorder.startRecording()
                        for await _ in clock.timer(interval: .seconds(1)) {
                            await send(.timerTicked)
                        }
                    },
                    .run { send in
                        for await level in audioRecorder.audioLevels() {
                            await send(.audioLevelUpdated(level))
                        }
                    }
                )
            case .stopRecording:
                state.isRecording = false
                return .run { send in
                    let fileId = try await audioRecorder.stopRecording()
                    await send(.recordingSaved(fileId))
                }
            case .timerTicked:
                state.duration += 1
                return .none
            case .audioLevelUpdated(let level):
                state.audioLevels.append(level)
                if state.audioLevels.count > 50 {
                    state.audioLevels.removeFirst()
                }
                return .none
            case .recordingSaved:
                return .send(.dismiss)
            case .recordingFailed:
                return .send(.dismiss)
            case .dismiss:
                return .none // 親がハンドル
            }
        }
    }
}
```

```swift
@Reducer
struct AskAIReducer {
    @ObservableState
    struct State: Equatable {
        let scope: ChatScope
        var messages: [ChatMessage] = []
        var inputText: String = ""
        var isGenerating: Bool = false
        var suggestions: [String] = [
            "要点をまとめて",
            "決定事項を確認",
            "次のアクションは？",
            "参加者の発言を分析"
        ]
        var selectedModel: String = "local"
        var isVoiceInputActive: Bool = false
    }

    struct ChatMessage: Equatable, Identifiable {
        let id: UUID
        let role: Role
        var content: String
        let timestamp: Date

        enum Role: Equatable {
            case user
            case assistant
        }
    }

    enum Action {
        case inputTextChanged(String)
        case sendTapped
        case suggestionTapped(String)
        case responseReceived(String)
        case responseStreamChunk(String)
        case responseFailed(String)
        case modelSwitchTapped
        case modelSelected(String)
        case voiceInputTapped
        case voiceInputResult(String)
        case attachTapped
    }

    @Dependency(\.llmRouter) var llmRouter

    var body: some ReducerOf<Self> {
        Reduce { state, action in
            switch action {
            case .inputTextChanged(let text):
                state.inputText = text
                return .none
            case .sendTapped:
                let text = state.inputText.trimmingCharacters(in: .whitespacesAndNewlines)
                guard !text.isEmpty else { return .none }
                let userMessage = ChatMessage(
                    id: UUID(), role: .user, content: text, timestamp: Date()
                )
                state.messages.append(userMessage)
                state.inputText = ""
                state.isGenerating = true
                state.suggestions = []

                let scope = state.scope
                let model = state.selectedModel
                let history = state.messages

                return .run { send in
                    let response = try await llmRouter.chat(
                        scope: scope,
                        model: model,
                        messages: history
                    )
                    await send(.responseReceived(response))
                } catch: { error, send in
                    await send(.responseFailed(error.localizedDescription))
                }
            case .suggestionTapped(let text):
                state.inputText = text
                return .send(.sendTapped)
            case .responseReceived(let content):
                state.isGenerating = false
                let aiMessage = ChatMessage(
                    id: UUID(), role: .assistant, content: content, timestamp: Date()
                )
                state.messages.append(aiMessage)
                return .none
            case .responseFailed(let error):
                state.isGenerating = false
                // Show error in chat
                let errorMessage = ChatMessage(
                    id: UUID(), role: .assistant,
                    content: "エラーが発生しました: \(error)",
                    timestamp: Date()
                )
                state.messages.append(errorMessage)
                return .none
            default:
                return .none
            }
        }
    }
}
```

### 6.3 AudioChunker

長時間ファイルの分割を担当する。

```swift
// AudioChunker.swift
import AVFoundation

protocol AudioChunkerProtocol: Sendable {
    func analyzeAndChunk(fileURL: URL) async throws -> [AudioChunk]
}

struct AudioChunk: Sendable {
    let index: Int
    let startSec: Double
    let endSec: Double
    let url: URL
}

final class AudioChunker: AudioChunkerProtocol {
    private let shortThreshold: TimeInterval = 60 * 60       // 60min
    private let longThreshold: TimeInterval = 60 * 60 * 3    // 180min
    private let standardChunkDuration: TimeInterval = 10 * 60 // 10min
    private let smallChunkDuration: TimeInterval = 5 * 60     // 5min

    func analyzeAndChunk(fileURL: URL) async throws -> [AudioChunk] {
        let asset = AVURLAsset(url: fileURL)
        let duration = try await asset.load(.duration).seconds

        if duration < shortThreshold {
            // 60分未満: 分割なし
            return [AudioChunk(index: 0, startSec: 0, endSec: duration, url: fileURL)]
        }

        let chunkDuration = duration < longThreshold
            ? standardChunkDuration  // 60-180min: 10分
            : smallChunkDuration     // 180min超: 5分

        var chunks: [AudioChunk] = []
        var start: Double = 0
        var index = 0

        while start < duration {
            let end = min(start + chunkDuration, duration)
            let chunkURL = try await exportChunk(
                from: asset, start: start, end: end, index: index
            )
            chunks.append(AudioChunk(
                index: index, startSec: start, endSec: end, url: chunkURL
            ))
            start = end
            index += 1
        }

        return chunks
    }

    private func exportChunk(
        from asset: AVURLAsset,
        start: Double,
        end: Double,
        index: Int
    ) async throws -> URL {
        let exportURL = FileManager.memora.chunksDirectory
            .appendingPathComponent("chunk_\(index)_\(UUID().uuidString).m4a")

        let exportSession = AVAssetExportSession(
            asset: asset,
            presetName: AVAssetExportPresetAppleM4A
        )!
        exportSession.outputURL = exportURL
        exportSession.outputFileType = .m4a
        exportSession.timeRange = CMTimeRange(
            start: CMTime(seconds: start, preferredTimescale: 600),
            end: CMTime(seconds: end, preferredTimescale: 600)
        )
        await exportSession.export()

        guard exportSession.status == .completed else {
            throw ChunkError.exportFailed(exportSession.error)
        }
        return exportURL
    }
}
```

### 6.4 TranscriptionEngine

```swift
// TranscriptionEngine.swift
import Speech

protocol TranscriptionEngineProtocol: Sendable {
    func transcribe(audioURL: URL, language: String?) async throws -> TranscriptionResult
}

struct TranscriptionResult: Sendable {
    let fullText: String
    let segments: [SegmentResult]

    struct SegmentResult: Sendable {
        let text: String
        let startSec: Double
        let endSec: Double
        let speakerLabel: String?
    }
}

final class TranscriptionEngine: TranscriptionEngineProtocol {
    func transcribe(audioURL: URL, language: String?) async throws -> TranscriptionResult {
        let locale = language.map { Locale(identifier: $0) } ?? Locale.current
        guard let recognizer = SFSpeechRecognizer(locale: locale),
              recognizer.isAvailable else {
            throw TranscriptionError.recognizerUnavailable
        }

        let request = SFSpeechURLRecognitionRequest(url: audioURL)
        request.shouldReportPartialResults = false
        request.addsPunctuation = true

        let result = try await withCheckedThrowingContinuation {
            (continuation: CheckedContinuation<SFSpeechRecognitionResult, Error>) in
            recognizer.recognitionTask(with: request) { result, error in
                if let error = error {
                    continuation.resume(throwing: error)
                } else if let result = result, result.isFinal {
                    continuation.resume(returning: result)
                }
            }
        }

        let segments = result.bestTranscription.segments.map { segment in
            TranscriptionResult.SegmentResult(
                text: segment.substring,
                startSec: segment.timestamp,
                endSec: segment.timestamp + segment.duration,
                speakerLabel: nil // SFSpeech doesn't provide speaker diarization natively
            )
        }

        return TranscriptionResult(
            fullText: result.bestTranscription.formattedString,
            segments: segments
        )
    }
}
```

話者分離（Speaker Diarization）は Phase 1 では SFSpeechRecognizer の標準機能では限定的であるため、文字起こし後に LLM を用いて推定するか、将来的にサードパーティ SDK を導入する余地を残す。Phase 1 では仮に全テキストを「Speaker 1」として格納し、将来の拡張に備える。

### 6.6 PipelineCoordinator

生成パイプライン全体を統括するサービスである。

```swift
// PipelineCoordinator.swift

protocol PipelineCoordinatorProtocol: Sendable {
    func execute(
        fileId: UUID,
        template: MeetingNote.TemplateType,
        model: String
    ) -> AsyncStream<PipelineEvent>
}

enum PipelineEvent: Sendable {
    case stepStarted(ProcessingJob.JobStep)
    case stepCompleted(ProcessingJob.JobStep)
    case chunkProgress(current: Int, total: Int)
    case completed
    case failed(step: ProcessingJob.JobStep, error: String)
}

final class PipelineCoordinator: PipelineCoordinatorProtocol {
    @Dependency(\.audioFileRepository) var audioFileRepository
    @Dependency(\.audioChunker) var audioChunker
    @Dependency(\.transcriptionEngine) var transcriptionEngine
    @Dependency(\.llmRouter) var llmRouter
    @Dependency(\.jobRepository) var jobRepository
    @Dependency(\.transcriptRepository) var transcriptRepository
    @Dependency(\.meetingNoteRepository) var meetingNoteRepository
    @Dependency(\.todoRepository) var todoRepository

    func execute(
        fileId: UUID,
        template: MeetingNote.TemplateType,
        model: String
    ) -> AsyncStream<PipelineEvent> {
        AsyncStream { continuation in
            Task {
                do {
                    guard let file = try await audioFileRepository.fetch(id: fileId) else {
                        continuation.yield(.failed(step: .transcription, error: "File not found"))
                        continuation.finish()
                        return
                    }

                    let fileURL = FileManager.memora.audioDirectory
                        .appendingPathComponent(file.localPath)

                    // Create job
                    let job = ProcessingJob(
                        templateType: template.rawValue,
                        modelProvider: model,
                        status: .running,
                        step: .transcription
                    )
                    job.audioFile = file
                    try await jobRepository.save(job)

                    // Step 1: Transcription
                    continuation.yield(.stepStarted(.transcription))
                    let chunks = try await audioChunker.analyzeAndChunk(fileURL: fileURL)

                    // Create chunk records
                    for chunk in chunks {
                        let chunkRecord = ProcessingChunk(
                            chunkIndex: chunk.index,
                            timeRangeStartSec: chunk.startSec,
                            timeRangeEndSec: chunk.endSec
                        )
                        chunkRecord.job = job
                        try await jobRepository.saveChunk(chunkRecord)
                    }

                    // Transcribe each chunk
                    var transcriptParts: [(Int, TranscriptionResult)] = []
                    for (i, chunk) in chunks.enumerated() {
                        let result = try await transcriptionEngine.transcribe(
                            audioURL: chunk.url, language: nil
                        )
                        transcriptParts.append((chunk.index, result))
                        continuation.yield(.chunkProgress(current: i + 1, total: chunks.count))
                    }

                    // Merge transcripts
                    transcriptParts.sort { $0.0 < $1.0 }
                    let fullText = transcriptParts.map { $0.1.fullText }.joined(separator: "\n\n")
                    let allSegments = transcriptParts.flatMap { pair in
                        pair.1.segments.map { seg in
                            Transcript.SpeakerSegment(
                                speakerLabel: seg.speakerLabel ?? "Speaker 1",
                                startSec: seg.startSec,
                                endSec: seg.endSec,
                                text: seg.text
                            )
                        }
                    }

                    let transcript = Transcript(
                        fullText: fullText,
                        language: Locale.current.language.languageCode?.identifier ?? "en",
                        speakerSegments: allSegments
                    )
                    transcript.audioFile = file
                    try await transcriptRepository.save(transcript)
                    continuation.yield(.stepCompleted(.transcription))

                    // Step 2: Summary
                    continuation.yield(.stepStarted(.summary))
                    let summaryResult = try await llmRouter.summarize(
                        transcript: fullText, template: template, model: model
                    )
                    continuation.yield(.stepCompleted(.summary))

                    // Step 3: Decisions
                    continuation.yield(.stepStarted(.decisions))
                    let decisions = try await llmRouter.extractDecisions(
                        transcript: fullText, model: model
                    )
                    continuation.yield(.stepCompleted(.decisions))

                    // Step 4: Todos
                    continuation.yield(.stepStarted(.todos))
                    let extractedTodos = try await llmRouter.extractTodos(
                        transcript: fullText, model: model
                    )
                    continuation.yield(.stepCompleted(.todos))

                    // Save MeetingNote
                    let note = MeetingNote(
                        templateType: template,
                        summary: summaryResult.summary,
                        decisions: decisions,
                        actionItems: summaryResult.actionItems,
                        generatedByModel: model
                    )
                    note.audioFile = file
                    try await meetingNoteRepository.save(note)

                    // Save Todos
                    for todo in extractedTodos {
                        let item = TodoItem(
                            title: todo.title,
                            sourceFileId: fileId,
                            sourceFileTitle: file.title,
                            projectId: file.project?.id,
                            assigneeLabel: todo.assigneeLabel
                        )
                        try await todoRepository.save(item)
                    }

                    // Update summaryLine on file
                    file.summaryLine = String(summaryResult.summary.prefix(80))
                    try await audioFileRepository.save(file)

                    // Mark job complete
                    job.status = .completed
                    job.updatedAt = Date()
                    try await jobRepository.save(job)

                    continuation.yield(.completed)
                    continuation.finish()
                } catch {
                    continuation.yield(.failed(
                        step: .transcription,
                        error: error.localizedDescription
                    ))
                    continuation.finish()
                }
            }
        }
    }
}
```

### 6.7 APIキーストア

```swift
// APIKeyStore.swift
import Security

protocol APIKeyStoreProtocol: Sendable {
    func save(provider: LLMProviderType, key: String) throws
    func retrieve(provider: LLMProviderType) throws -> String?
    func delete(provider: LLMProviderType) throws
}

enum LLMProviderType: String, CaseIterable, Sendable {
    case openAI    = "openai"
    case anthropic = "anthropic"
    case google    = "google"
    case deepSeek  = "deepseek"
}

final class APIKeyStore: APIKeyStoreProtocol {
    private let serviceName = "com.memora.apikeys"

    func save(provider: LLMProviderType, key: String) throws {
        let data = Data(key.utf8)
        let query: [String: Any] = [
            kSecClass as String: kSecClassGenericPassword,
            kSecAttrService as String: serviceName,
            kSecAttrAccount as String: provider.rawValue,
            kSecValueData as String: data
        ]
        SecItemDelete(query as CFDictionary)
        let status = SecItemAdd(query as CFDictionary, nil)
        guard status == errSecSuccess else {
            throw KeychainError.saveFailed(status)
        }
    }

    func retrieve(provider: LLMProviderType) throws -> String? {
        let query: [String: Any] = [
            kSecClass as String: kSecClassGenericPassword,
            kSecAttrService as String: serviceName,
            kSecAttrAccount as String: provider.rawValue,
            kSecReturnData as String: true,
            kSecMatchLimit as String: kSecMatchLimitOne
        ]
        var result: AnyObject?
        let status = SecItemCopyMatching(query as CFDictionary, &result)
        guard status == errSecSuccess, let data = result as? Data else {
            return nil
        }
        return String(data: data, encoding: .utf8)
    }

    func delete(provider: LLMProviderType) throws {
        let query: [String: Any] = [
            kSecClass as String: kSecClassGenericPassword,
            kSecAttrService as String: serviceName,
            kSecAttrAccount as String: provider.rawValue
        ]
        SecItemDelete(query as CFDictionary)
    }
}
```

```swift
@Reducer
struct AppReducer {
    @ObservableState
    struct State: Equatable {
        var hasCompletedOnboarding: Bool = false
        var authState: AuthReducer.State = AuthReducer.State()
        var subscriptionStatus: SubscriptionStatus = .free
        var showPaywall: Bool = false
        var selectedTab: Tab = .files

        // Tab states
        var filesState: FilesListReducer.State = FilesListReducer.State()
        var projectsState: ProjectsListReducer.State = ProjectsListReducer.State()
        var todoState: TodoListReducer.State = TodoListReducer.State()
        var settingsState: SettingsReducer.State = SettingsReducer.State()

        // Flow states
        var onboardingState: OnboardingReducer.State? = OnboardingReducer.State()
        var paywallState: PaywallReducer.State? = nil

        enum Tab: String, CaseIterable {
            case files    = "Files"
            case projects = "Projects"
            case todo     = "ToDo"
            case settings = "Settings"
        }
    }

    enum Action {
        case onLaunch
        case tabSelected(State.Tab)
        case onboarding(OnboardingReducer.Action)
        case auth(AuthReducer.Action)
        case paywall(PaywallReducer.Action)
        case files(FilesListReducer.Action)
        case projects(ProjectsListReducer.Action)
        case todo(TodoListReducer.Action)
        case settings(SettingsReducer.Action)
        case subscriptionStatusChanged(SubscriptionStatus)
    }

    var body: some ReducerOf<Self> {
        Scope(state: \.authState, action: \.auth) { AuthReducer() }
        Scope(state: \.filesState, action: \.files) { FilesListReducer() }
        Scope(state: \.projectsState, action: \.projects) { ProjectsListReducer() }
        Scope(state: \.todoState, action: \.todo) { TodoListReducer() }
        Scope(state: \.settingsState, action: \.settings) { SettingsReducer() }

        Reduce { state, action in
            switch action {
            case .onLaunch:
                let completed = UserDefaults.standard.bool(forKey: "hasCompletedOnboarding")
                state.hasCompletedOnboarding = completed
                if completed {
                    state.onboardingState = nil
                }
                return .send(.auth(.onAppear))

            case .onboarding(.completed):
                state.hasCompletedOnboarding = true
                UserDefaults.standard.set(true, forKey: "hasCompletedOnboarding")
                state.onboardingState = nil
                // After onboarding, show auth if not logged in
                return .none

            case .auth(.authStateChanged(let user)):
                if user != nil && !state.showPaywall {
                    // First login → show paywall
                    let shownBefore = UserDefaults.standard.bool(forKey: "hasShownPaywall")
                    if !shownBefore {
                        state.showPaywall = true
                        state.paywallState = PaywallReducer.State()
                        UserDefaults.standard.set(true, forKey: "hasShownPaywall")
                    }
                }
                return .none

            case .paywall(.dismissed), .paywall(.purchased):
                state.showPaywall = false
                state.paywallState = nil
                return .none

            case .tabSelected(let tab):
                state.selectedTab = tab
                return .none

            case .subscriptionStatusChanged(let status):
                state.subscriptionStatus = status
                return .none

            default:
                return .none
            }
        }
        .ifLet(\.onboardingState, action: \.onboarding) { OnboardingReducer() }
        .ifLet(\.paywallState, action: \.paywall) { PaywallReducer() }
    }
}
```

```swift
enum MemoraError: LocalizedError {

    case microphonePermissionDenied
    case recordingFailed(underlying: Error)

        }
```

### 9.2 リトライ方針

パイプラインのチャンク処理では、各チャンクに最大3回のリトライを許可する。リトライ間隔は指数バックオフ（1秒、3秒、9秒）とする。3回失敗した場合、そのチャンクを `failed` に設定し、他のチャンクの処理は継続する。全チャンク完了後、失敗チャンクがある場合はジョブ全体を `failed` にし、ユーザーに部分的な再試行を提案する。

LLM リクエストでは HTTP 429（Rate Limit）を受けた場合、`Retry-After` ヘッダーの値に従って待機する。HTTP 5xx の場合は3回まで指数バックオフでリトライする。HTTP 4xx（429以外）はリトライせずにエラーを返す。

### 9.3 パーミッション管理

マイクと音声認識のパーミッションは、録音開始前と文字起こし開始前にそれぞれチェックする。未許可の場合は iOS 標準のパーミッションダイアログを表示する。拒否済みの場合は設定アプリへの誘導アラートを表示する。

```swift
enum PermissionManager {
    static func requestMicrophonePermission() async -> Bool {
        await withCheckedContinuation { continuation in
            AVAudioApplication.requestRecordPermission { granted in
                continuation.resume(returning: granted)
            }
        }
    }

    static func requestSpeechRecognitionPermission() async -> Bool {
        await withCheckedContinuation { continuation in
            SFSpeechRecognizer.requestAuthorization { status in
                continuation.resume(returning: status == .authorized)
            }
        }
    }
}
```

## 第12部 非機能要件

### 12.1 パフォーマンス

Files 一覧は 1000 件のファイルでもスムーズにスクロールできるように、SwiftData の `FetchDescriptor` に `fetchLimit` と `fetchOffset` を設定したページネーションを実装する。1回のフェッチで 50 件を取得し、リスト末尾到達時に次のバッチをロードする。

録音中の AudioLevel の更新は 0.05 秒間隔で行うが、UI 更新は 0.1 秒間隔に間引く。

文字起こしと LLM リクエストは常にバックグラウンドタスクとして実行し、メインスレッドをブロックしない。

### 12.2 アクセシビリティ

すべてのインタラクティブ要素に `accessibilityLabel` を設定する。Dynamic Type の全サイズに対応する。VoiceOver でのナビゲーションが可能であることを各画面で確認する。色のみに依存する情報表現は行わず、アイコンやテキストを併用する。

### 12.3 ローカライゼーション

Phase 1 では日本語をプライマリ、英語をセカンダリとする。すべてのユーザー向け文字列は `Localizable.xcstrings` で管理し、ハードコードしない。日付フォーマットは `Date.FormatStyle` を使用してロケール適応させる。

### 12.4 セキュリティ

APIキーは Keychain に保存し、UserDefaults やファイルには書き出さない。音声ファイルと添付ファイルは Application Support ディレクトリに保存し、iTunes バックアップ対象外フラグ（`isExcludedFromBackup`）は Phase 1 では設定しない（クラウドバックアップは Phase 2 で検討）。ネットワーク通信は HTTPS のみを使用する。
