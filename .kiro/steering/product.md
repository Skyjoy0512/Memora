# Memora - Product Overview

## Product Summary
Memora は iPhone で議事録、文字起こし、要約を簡単に行える iOS アプリです。会議、講義、インタビューなど、様々な場面での記録作業を効率化します。

## Key Features

### Core Features
- **議事録作成**: 会議の記録を簡単に管理
- **録音機能**: 音声をクリアに録音（AVFoundation）
- **文字起こし**: 音声をテキストに変換
- **要約**: 長い会議の内容を要約
- **話者分離**: Speaker 1 / Speaker 2 などのラベル付け（初期は仮ラベル）

### Future Features
- **Ask AI**: 文字起こし内容に AI で質問
- **添付資料**: 資料を録音と紐付け
- **エクスポート**: テキスト、PDF 形式で出力
- **iCloud 同期**: デバイス間で同期

## Target Users
- ビジネスパーソン（会議記録）
- 学生（講義録音）
- インタビュアー（インタビュー記録）

## Value Proposition
- シンプルで洗練された UI で直感的に操作可能
- iOS ネイティブの操作感で自然に使える
- モノトーン基調でプロフェッショナルなデザイン

## UX Principles
- **エラー表示**: 軽いトースト表示で伝える
- **情報の階層を明確に**: 重要な情報が目立つように
- **iOSらしい滑らかさ**: アニメーション、トランジションを自然に

## Key User Flows
1. **録音フロー**: Home 画面 → 録音開始 → 録音停止 → 保存
2. **文字起こしフロー**: ファイル選択 → 文字起こし開始 → 完了 → 要約生成
3. **議事録閲覧**: Project 選択 → File 選択 → 要約・文字起こし閲覧
