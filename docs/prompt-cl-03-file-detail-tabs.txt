# 6. 実装方針
このタスクは **大部分が実装済み**。残存する改善点を確認し、あれば最小差分で対応:

1. **確認すべき点**:
   - `showTranscriptView` / `showSummaryView` プロパティが ViewModel に存在するが使われていないか？ → 使われていれば削除またはタブ切り替えに置換
   - `GenerationFlowSheet` の起動方法がタブ内から可能か → `vm.showGenerationFlow = true` で summaryTab からボタンで起動可能
   - 要約未実行時に「文字起こし結果を表示」「要約結果を表示」のような別画面遷移ボタンが残っていないか

2. **改善が必要な場合**:
   - 要約生成フローのトリガーを summaryTab 内に統合（既にボタンあり）
   - 文字起こし開始トリガーを transcriptTab 内に統合（既にボタンあり）
   - ViewModel の `showTranscriptView / showSummaryView` が未使用なら削除

3. **受け入れ条件を満たす最小差分**:
   - タブ UI が常設で機能している → ✅
   - memo の autosave / index rebuild が壊れていない → ✅
   - 画面遷移の往復が減っている → ✅

**結論**: 現状で受け入れ条件を満たしている場合は、CL-03 を DONE とマークし、改善点があれば最小差分で追加する。
