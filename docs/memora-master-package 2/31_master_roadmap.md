## 3. 3つの並行トラック

実装リソースが複数あるなら、この3トラックを並行できる:

```
トラック STT(1人が連続で):  14 → 02 → 07 → 06/03 のSTT部分
トラック UI(別担当):        01 → 04 → 05 → 03/06 のUI部分 → 22 のUI
トラック デスクトップ(別):   21実証 → macOS雛形 → (Core切り出し待ち) → 会議録音
```

デバイス連携(フェーズ3)は STT/UI が一段落してから、または3人目がいれば並行。
