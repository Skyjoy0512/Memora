## 3つの要望への対応まとめ

### 要望1「出力が差分だけ? 更新パッケージが欲しい」
→ **これが統合済みの最新パッケージ**です。以前の2パッケージの全内容 + 新規2本(14/24)+ 更新ロードマップ(31)を1つにまとめました。以前の個別 zip はもう不要です。

### 要望2「デスクトップアプリを並行して作りたい」
→ **`24_desktop_parallel_dev.md`** に、iOS を止めずに並行開発する具体策を追加。鍵は Core 層の SPM 共有パッケージ化と段階移行。ScreenCaptureKit 実証と macOS 雛形は**今すぐ並行着手できる**部分として明記。

### 要望3「長時間ファイルが重い・クラッシュする」
→ **`14_long_file_streaming.md`** で根本原因(全チャンク一括書き出し+全結果メモリ保持による OS のメモリ強制終了)を特定し、ストリーミング処理(1個ずつ作って処理して即削除)で解決。**最優先の P0**として 31 の先頭に配置。
