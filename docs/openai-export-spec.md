# OpenAI Export 仕様・制限事項

## 概要

Memora の OpenAI Export 機能は、会議の文字起こし・要約データを OpenAI Files API にアップロードし、ChatGPT で参照可能にする。

## 対応フォーマット

| フォーマット | 拡張子 | 用途 |
|-------------|--------|------|
| JSON | `.json` | 構造化データ。Assistants API / プログラム利用向け |
| Markdown | `.md` | 人間可読。ChatGPT の添付ファイルとして即座に参照可能 |

## アップロード先

| Purpose | 説明 |
|---------|------|
| `user_data` | ChatGPT のファイル添付として利用（デフォルト） |
| `assistants` | Assistants API の file search / code interpreter で利用 |

## 制限事項

| 制限 | 詳細 |
|------|------|
| **ChatGPT memory 直接書き込み不可** | OpenAI は ChatGPT のパーソナルメモリへの書き込み API を公開していない。アップロードしたファイルはファイル添付であり、永続的なメモリではない。会話内で明示的に参照する必要がある。 |
| **ファイルサイズ上限** | 512 MB / ファイル。会議の文字起こし・要約は通常 1 MB 未満。 |
| **API 課金** | ファイルストレージは無料だが、ファイルを会話で利用するとトークン消費が発生する。OpenAI API の標準料金が適用される。 |
| **手動エクスポート** | エクスポートは手動の一回限りの操作。エクスポート後の Memora 側のデータ変更は反映されない。再エクスポートが必要。 |
| **ファイル有効期限** | アップロードされたファイルは API または OpenAI ダッシュボードから明示的に削除されるまで保持される。自動削除なし。 |
| **ChatGPT 会話の自動作成不可** | ファイルのアップロードのみ。ChatGPT の会話の作成やメッセージ送信は行わない。ChatGPT を開いてファイルを参照する必要がある。 |
| **API キー必須** | 設定画面で OpenAI API キーを入力済みであること。Files API は課金有効なアカウントでのみ利用可能。 |
| **レート制限** | OpenAI のレート制限が適用される。大量の一括エクスポートは制限される場合がある。 |
| **ネットワーク必須** | エクスポートにはインターネット接続が必要。 |

## プライバシーに関する注意事項

- 文字起こし・要約データが OpenAI サーバーに送信される
- エクスポート前にデータ内容を確認することを推奨
- アップロードしたファイルは OpenAI ダッシュボードまたは API から削除可能
- Memora のプライバシー設定はこのエクスポートに影響しない（ユーザーが明示的に選択）

## 今後の改善予定

- データ変更時の自動再エクスポート（オプション）
- プロジェクト単位の一括エクスポート
- MCP サーバーモード（`docs/mcp-server-spec.md` 参照）
- ChatGPT Conversations API 対応（OpenAI が公開した場合）
- エクスポート履歴の永続化
