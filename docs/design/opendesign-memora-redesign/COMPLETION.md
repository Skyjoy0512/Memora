## 主要な設計判断（再掲）

1. アクセント: `#FF3030`（赤）→ `#1A7F6B`（深い青緑） — 知識アプリに落ち着きと信頼を
2. キャプチャ導線: 3段階モーダル → 1画面完結 + 自動保存 + バックグラウンド処理
3. ホーム: 3モードドロップダウン → セグメント + 単一タイムライン + 常時検索バー
4. ファイル詳細: 3タブ → 4タブ（概要/文字起こし/メモ/質問）
5. 視覚的差別化: 紙とインクの質感 / 余白のリズム / シグネチャーモーション「開く」

## 実装開始手順

1. CODEX（またはClaude Code実装エージェント）に `CODEX_IMPLEMENTATION_PROMPT.md` を読み込ませる
2. Phase 1（デザイントークン置換）から順に実行
3. 各Phase完了時に `npm run typecheck` で検証
4. Phase 8 の全チェック項目（60項目）が満たされた時点で実装完了

**実装の基点**: `apps/mobile-expo/src/design/tokens.ts` の書き換え（DESIGN.md §2-8 の新トークンに置換）
