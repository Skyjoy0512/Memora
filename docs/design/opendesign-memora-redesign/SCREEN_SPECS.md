## 画面 1: ホーム

**ファイル**: `src/screens/HomeScreen.tsx`
**ルート**: `app/(tabs)/index.tsx`

### 目的

ユーザーがアプリを開いたとき、最も重要な情報（最近の記録）に即座にアクセスでき、新しい記録を開始する導線が明確であること。

### 情報階層

```
┌──────────────────────────────┐
│ ステータスバー                  │
├──────────────────────────────┤
│ [ユーザー名 / 日付]   [設定] [通知] │  ← ヘッダー（スクロールで隠れる）
├──────────────────────────────┤
│ 🔍 記録を検索                  │  ← 検索バー（常時表示、フォーカスで拡大）
├──────────────────────────────┤
│ ┌──────────────────────────┐ │
│ │ 📝 最近の記録              │ │  ← セクションヘッダー
│ │ ┌────────────────────┐   │ │
│ │ │ 定例ミーティング     │   │ │  ← ファイルカード
│ │ │ 今日 · 32分 · 要約済  │   │ │
│ │ └────────────────────┘   │ │
│ │ ┌────────────────────┐   │ │
│ │ │ アイデアメモ         │   │ │
│ │ │ 昨日 · 5分 · 文字起こし中│   │ │
│ │ └────────────────────┘   │ │
│ └──────────────────────────┘ │
│                              │
│ ── 7月10日（木）──             │  ← 日付セパレーター
│ ┌────────────────────┐       │
│ │ ...                     │ │
│                              │
│ [空状態: 最初の録音を始める]      │  ← ファイル0件時のみ
└──────────────────────────────┘
```

### セグメントコントロール

画面タイトル部にセグメントコントロールを配置：

| セグメント | 表示内容 |
|-----------|---------|
| すべて | 全ファイルを時系列で表示（デフォルト） |
| お気に入り | スターをつけたファイルのみ |
| プロジェクト | プロジェクト一覧カードグリッド |

### ファイルカード

各ファイルは以下の情報を持つカードとして表示：

```
┌─────────────────────────────────────┐
│ 🎙 音声  │ 定例ミーティング            │  ← アイコン + タイトル
│          │ 今日 14:30 · 32分          │  ← 日時・長さ
│          │ [要約済]                   │  ← ステータスピル
│          │ 決定事項：次回までに資料を…   │  ← 要約の先頭1行
│          │                     [ ⋯ ]  │  ← その他メニュー
└─────────────────────────────────────┘
```

### 表示条件と状態

| 状態 | 条件 | 表示 |
|------|------|------|
| 通常 | ファイルあり | ファイルカードリスト |
| 初回空 | `files.length === 0 && !isLoading` | 空状態（イラスト + CTA） |
| 検索空 | `searchQuery && filteredFiles.length === 0` | 「一致する記録はありません」 |
| 読み込み中 | `isLoading && files.length === 0` | スケルトンカード ×5 |
| 更新中 | `isRefreshing` | リスト上部に小さなインジケータ |
| エラー | `error` | エラー状態 + 再試行ボタン |
| オフライン | ネットワーク未接続 | バナー「オフライン — 端末内の記録のみ表示」 |

### 操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| ファイルを開く | カードタップ | `file/[id]` に遷移 |
| ファイルメニュー | カード右端の「⋯」タップ | アクションシート（名前変更・プロジェクト移動・削除） |
| 検索 | 検索バーに入力 | リアルタイムフィルタリング |
| プルダウン更新 | リスト上部でプルダウン | `refresh({ silent: true })` |
| セグメント切替 | セグメントコントロールタップ | ビュー切替 |
| 録音開始 | タブバーFAB →「録音」 | 録音モーダルを表示 |
| 設定へ | ヘッダー設定アイコン | `/settings` に遷移 |

### 空状態

**初回空状態（ファイル0件）**:
```
        🎙（大きめのアイコン）
    最初の記録を残してみましょう

    取り込みを始められます

```

## 画面 2: 録音

**ファイル**: `src/features/capture/CaptureFlowProvider.tsx`（改修）
**表示**: FABタップ → モーダル

### 目的

録音を開始してから保存完了まで、1画面で完結する。ユーザーは操作を意識せず、自然に記録を残せる。

### 録音中画面

```
┌──────────────────────────────┐
│ [閉じる]              [破棄]  │  ← ヘッダー
├──────────────────────────────┤
│                              │
│           ● 録音中             │  ← ステータス（赤いドットが点滅）
│           12:34              │  ← 経過時間（モノスペース・大）
│                              │
│   ▏▎▍▌▋▊▉█▉▊▋▌▍▎▏           │  ← 波形（リアルタイム）
│                              │
│  ┌────────────────────────┐  │
│  │ ひと言メモ（あとで編集できます）│  │  ← クイックメモ入力欄
│  └────────────────────────┘  │
│                              │
│  ┌────────────────────────┐  │
│  │ タイトル: 新しい録音        │  │  ← タイトル入力（デフォルトは日時）
│  └────────────────────────┘  │
│                              │
│  [テンプレート: 議事録 ▾]      │  │  ← テンプレート選択（省略可）
│                              │
│       ◼ 停止                  │  ← 停止ボタン（中央・大）
│                              │
└──────────────────────────────┘
```

### 録音停止後

```
┌──────────────────────────────┐
│          ✓ 保存しました         │  ← 完了表示
│                              │
│   文字起こしと要約を自動で        │
│   作成しています              │
│                              │
│   [表示]    [ホームに戻る]     │  ← アクション
└──────────────────────────────┘
```

※ 録音停止後、自動的にホームに戻る。文字起こし・要約はバックグラウンドで実行され、Dynamic Island Pill に進捗が表示される。

### 表示条件と状態

| 状態 | 条件 | 表示 |
|------|------|------|
| 録音準備 | `mode === 'preparing'` | マイク権限チェック中 |
| 録音中 | `mode === 'recording'` | 上記レイアウト |
| 一時停止 | `mode === 'recording' && isPaused` | 波形が灰色に、経過時間が点滅 |
| 保存中 | `mode === 'saving'` | スピナー + 「保存しています」 |
| 保存完了 | `mode === 'completed'` | チェックマーク + 「保存しました」 |
| 権限未許可 | マイク権限なし | 権限リクエスト画面 |
| エラー | 保存失敗 | エラー表示 + 「再試行」「破棄」 |

### 操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| 停止 | 停止ボタンタップ | 録音停止 → 自動保存 → バックグラウンド処理開始 |
| 破棄 | 右上「破棄」タップ → 確認 | 録音データ破棄 → ホームに戻る |
| メモ入力 | メモ欄タップ | キーボード表示。録音中でも入力可能 |
| タイトル編集 | タイトル欄タップ | キーボード表示 |
| テンプレート選択 | テンプレート行タップ | テンプレート選択ボトムシート |
| 閉じる | 左上「閉じる」 | 録音は継続、ホームに戻る。Dynamic Island Pill に録音中表示 |

### 権限未許可状態

```
        🎤（大きめのアイコン）
    マイクへのアクセスを許可してください
  録音するにはマイクの使用許可が必要です。
  設定アプリから「マイク」をオンにしてください。
      [ 設定を開く ]（ボタン）
```

## 画面 3: ファイル詳細

**ファイル**: `src/screens/FileDetailScreen.tsx`
**ルート**: `app/file/[id].tsx`

### 目的

記録の内容を理解し、必要な情報に素早くアクセスできる。文字起こし・要約・自分のメモの3つの視点から記録を見られる。

### レイアウト

```
┌──────────────────────────────┐
│ [←]  定例ミーティング    [共有][⋯]│  ← ヘッダー
│       今日 14:30 · 32分        │
├──────────────────────────────┤
│  [概要] [文字起こし] [メモ] [質問] │  ← タブバー（4タブ）
├──────────────────────────────┤
│                              │
│  ┌──── タブ内容 ────┐         │
│  │                    │         │
│  │  （タブによって切替）  │         │
│  │                    │         │
│  └────────────────────┘         │
│                              │
└──────────────────────────────┘
│ [▶ 再生]  00:32 / 32:00  [1×] │  ← 再生バー（固定フッター）
```

### 表示条件と状態

| 状態 | 条件 | 表示 |
|------|------|------|
| 読み込み中 | `isLoading` | スケルトン詳細 |
| 存在しない | `!file && !isLoading` | 「この記録は削除されたか、存在しません」 |
| 通常 | ファイルあり | 上記レイアウト |
| 文字起こし前 | `transcript.length === 0` | 「文字起こしはまだありません」+ 開始ボタン |
| 文字起こし中 | `transcription.isRunning` | 進捗表示 + 「文字起こしを実行中…」 |
| 文字起こし失敗 | `transcription.error` | エラー表示 + 「再試行」 |
| 要約未生成 | `file.status !== 'summarized'` | 要約セクションに生成ボタン |
| 要約生成中 | `isGeneratingSummary` | スピナー + 「要約を作成中…」 |
| 要約失敗 | `summaryError` | エラーメッセージ + 「もう一度生成」 |
| 再生エラー | `playback.error` | 「再生できません」 |
| 削除確認中 | `isDeleteOpen` | 削除確認モーダル |

### 操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| 戻る | ヘッダー ← / スワイプ | 前画面に戻る |
| タブ切替 | タブタップ | タブコンテンツ切替（フェードアニメーション） |
| 再生/一時停止 | 再生バータップ | 音声再生 |
| シーク | 再生バードラッグ | 再生位置移動。文字起こしタブで該当セグメントに自動スクロール |
| 速度変更 | 再生バー速度ボタン | 1× → 1.5× → 2× → 0.75× → 1× |
| 文字起こし開始 | 「文字起こしを開始」ボタン | バックグラウンド処理開始 |
| 要約生成 | 「要約を生成」ボタン | バックグラウンド処理開始 |
| アクション完了 | チェックボックスタップ | タスクに完了マーク。タスク画面に反映 |
| メモ編集 | メモエリアタップ | 編集モードに（インライン編集） |
