Memora — UI/UX 全面刷新案 / 磨き込み v3 ・ PLAUDテイスト

画面遷移・ワイヤーフレーム・UIUXの再設計

① Liquid Glass を撤去し、Claude / PLAUD 系の平面モノトーンへ。ぼかし・光沢を全廃し、白カード＋薄い罫線・影のみの構成に統一。既存の MemoraColor トークン（背景 #ECECEC・本文 #0D0D0D・区切り線 #E5E5EA）はそのまま踏襲。
② ナビ構造を「記録の入口」中心に再編。タブは ホーム / タスク / Ask / 設定 の4つに整理し、ファイル/プロジェクトの切替はホーム内のセグメントに統合。検索バーを「Memoraに質問・検索」の兼用バーへ格上げし、AskAI をアプリ全体の主役として常時露出。
③ File Detail を「要約 / 文字起こし / メモ」の3タブ＋最優先で使いやすく。下部に AskAI 常設バーを固定し、ヘッダーの共有アイコンから Notion / ChatGPT / Markdown 等への書き出しシートに直結。
④ タスクは記録と紐付いた状態で表示。どの会議・ファイル由来のタスクかを常に明示し、ToDo の存在感を強化。
⑤ 今回の磨き込み。録音のライブ感（カウントアップ・波形・ストリーミング文字起こし）／生成の非同期プログレス／文字起こしのタイムスタンプ連携／Ask の実送信＋引用元チップ／タスクのグルーピング／空状態・シートアニメーション・押下フィードバックなど、実インタラクションを追加。
⑥ v3: PLAUD テイストへ再調整。純白背景・カード/枠線の撤去・余白とヘアラインだけの区切り・大見出しタイポ・下線タブ・録音画面のライト化・モノトーン徹底など、PLAUD アプリの空気感に全面を寄せました。

操作メモ（磨き込みポイント）
・録音画面: カウントアップ・波形反応・ストリーミング文字起こし・一時停止・×で破棄確認
・停止 → 生成シート（ファイル名編集・カスタムテンプレ・スキップ可）→ 生成中プログレス画面 → File Detail
・File Detail: シークバー操作・再生速度切替・文字起こしタップでシーク・決定事項/次のアクション構造化・タスク化で追加済み表示・メモは編集可能
・タスク: 期限切れ/今日/今後/完了でグルーピング、完了は折りたたみ、由来ファイルタップで元ファイルへ
・Ask: 実際に送信可能、送信中はスケルトン表示、回答に引用元チップ、新しい会話で空状態＋サジェスト表示
・設定: 通知トグル・言語・アカウント/プラン・データ削除は確認ダイアログ付き
・全シート: スライドアップアニメーション、トーストはフェード＋スライドで出現
・起動導線: オンボーディング（3枚）→ ログイン（Apple/Google/メール）→ Paywall（Pro訴求）→ ホーム。左下「オンボーディングから見る」でいつでも再生できます
対象外（プロトタイプの範囲外として意図的に省略）：実音声再生・実スワイプ操作（削除/完了）・Pull-to-refresh・背景録音の最小化・複数デバイスのスワイプ切替・話者名編集・文字起こし内検索。既存要素すべての角丸/フォントサイズの網羅的統一までは踏み込まず、新規/変更箇所のみ整理しています。
